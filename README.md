# Dashboard_Hospital-Emergency-Room

## Overview

Dashboard Hospital Emergency Room merupakan dashboard analisis layanan Instalasi Gawat Darurat (IGD) yang dikembangkan menggunakan Microsoft Power BI. Dashboard ini dirancang untuk membantu rumah sakit dalam memantau performa pelayanan pasien, mengidentifikasi pola kunjungan pasien, mengevaluasi waktu tunggu pelayanan, serta meningkatkan kualitas layanan berdasarkan data yang tersedia.

Melalui visualisasi interaktif, dashboard ini memungkinkan pihak manajemen rumah sakit untuk memperoleh informasi secara cepat dan akurat sehingga dapat mendukung pengambilan keputusan berbasis data (data-driven decision making)

## Project Objectives

Tujuan utama pengembangan dashboard ini adalah:
- Memantau jumlah kunjungan pasien ke Instalasi Gawat Darurat (Emergency Room).
- Menganalisis rata-rata waktu tunggu pasien sebelum mendapatkan pelayanan.
- Mengukur tingkat kepuasan pasien terhadap layanan rumah sakit.
- Mengetahui distribusi pasien berdasarkan kelompok usia dan jenis kelamin.
- Mengidentifikasi status pasien setelah mendapatkan pelayanan (admitted atau not admitted).
- Menganalisis departemen tujuan rujukan pasien.
- Menyediakan informasi yang mudah dipahami oleh manajemen rumah sakit melalui visualisasi data yang interaktif.

## Key Performance Indicators (KPIs)
Dashboard menampilkan beberapa indikator utama yang digunakan untuk mengevaluasi performa layanan Emergency Room, yaitu:
1. Total Number of Patients
Menampilkan jumlah keseluruhan pasien yang melakukan kunjungan ke Instalasi Gawat Darurat.

2. Average Wait Time
Menampilkan rata-rata waktu tunggu pasien sebelum mendapatkan pelayanan medis.

3. Patient Satisfaction Score
Menampilkan tingkat kepuasan pasien terhadap pelayanan rumah sakit.

4. Admission Status
Menampilkan perbandingan jumlah pasien yang dirawat (Admitted) dan tidak dirawat (Not Admitted).

5. Gender Distribution
Menampilkan distribusi pasien berdasarkan jenis kelamin.

6. Age Group Analysis
Menampilkan distribusi pasien berdasarkan kelompok usia untuk mengetahui segmen pasien yang paling sering menggunakan layanan IGD.

7. Department Referral Analysis
Menampilkan informasi mengenai departemen tujuan rujukan pasien setelah mendapatkan pelayanan awal.

## Dashboard Features
1. Patient Overview
Menyediakan ringkasan kondisi layanan rumah sakit secara keseluruhan melalui KPI utama.

2. Wait Time Analysis
Membantu mengidentifikasi potensi keterlambatan pelayanan dan area yang membutuhkan perbaikan.

3. Patient Satisfaction Analysis
Memberikan gambaran mengenai tingkat kepuasan pasien terhadap pelayanan yang diterima.

4. Demographic Analysis
Menganalisis karakteristik pasien berdasarkan usia dan jenis kelamin.

5. Admission Analysis
Membandingkan jumlah pasien yang dirawat dan tidak dirawat setelah pemeriksaan.

6.Referral Department Analysis
Menunjukkan departemen yang paling banyak menerima rujukan dari Emergency Room.

7. Interactive Filtering
Dashboard dilengkapi filter interaktif sehingga pengguna dapat melakukan eksplorasi data secara lebih mendalam.

## Business Insights
Dashboard ini dapat membantu rumah sakit dalam:
- Mengidentifikasi jam atau periode dengan jumlah pasien tertinggi.
- Mengetahui kelompok usia yang paling sering menggunakan layanan Emergency Room.
- Mengevaluasi kualitas pelayanan melalui analisis waktu tunggu pasien.
- Memantau tingkat kepuasan pasien.
- Menentukan departemen yang paling sering menerima rujukan.
- Mendukung peningkatan efisiensi operasional rumah sakit.
